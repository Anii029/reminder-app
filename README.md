# 🖥️ Reminder App

A modern desktop reminder application built using Python and CustomTkinter.

---

## 🚀 Features

* ⏰ Add reminders with time
* 🗑️ Delete reminders
* 🔔 Desktop notifications
* 💾 Saves reminders automatically
